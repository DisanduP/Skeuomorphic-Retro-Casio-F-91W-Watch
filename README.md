# Skeuomorphic Retro Casio F-91W Watch

A faithful digital recreation of the iconic Casio F-91W wristwatch, built entirely with pure HTML, CSS, and JavaScript. This project brings the retro aesthetic and functionality of the classic digital watch to the web.

## Features

- **Real-time Clock**: Displays current time, day, and weekday, updating every second.
- **Skeuomorphic Design**: Pixel-perfect recreation of the original Casio F-91W appearance, including the LCD screen and plastic case.
- **Interactive Elements**: Functional light button that simulates the backlight feature.
- **Responsive Layout**: Centered and scalable design suitable for web display.
- **No Images Required**: Entirely vector-based using CSS for a crisp, scalable look.

## Installation & Usage

1. Clone the repository:
   ```
   git clone https://github.com/BinaryVortex/Skeuomorphic-Retro-Casio-F-91W-Watch.git
   ```

2. Open `index.html` in your web browser.

No build process or dependencies required – it's a static website!

## How It Works

The watch face is constructed using HTML elements styled with CSS to mimic the appearance of an LCD display and plastic watch case. JavaScript handles:

- Fetching the current date and time using `Date()` object.
- Updating the DOM elements every second with `setInterval`.
- Simulating the light button press with event listeners and CSS classes.

The design is skeuomorphic, meaning it closely replicates the physical appearance and behavior of the real watch.
